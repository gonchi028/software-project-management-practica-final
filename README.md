# Resumen
---
El presente documento consolida la totalidad de la documentación de gestión del proyecto "Sistema de Gestión para Mercados Municipales", encargado por la Alcaldía de Sucre al equipo de desarrollo. El proyecto tiene por objetivo digitalizar la administración de los mercados Campesino, Central y Entre Ríos, abarcando el registro de puestos y comerciantes, cobros de tarifas, control de licencias sanitarias y generación de informes mensuales para la Dirección de Ingresos.

Se cuenta con un plazo de seis meses para entregar un Producto Mínimo Viable (MVP) funcional y un presupuesto referencial de 70.000 Bs. La metodología adoptada es híbrida: Scrum para el desarrollo iterativo del software, complementado con prácticas tradicionales (cascada) para la gobernanza contractual, control presupuestario y entregables formales que exige una entidad pública.

Durante la ejecución se presentaron tres situaciones críticas previstas en la planificación: (i) un cambio en la normativa municipal sobre licencias sanitarias a mitad del proyecto; (ii) la renuncia de dos desarrolladores clave, riesgo previamente identificado en la matriz con probabilidad media e impacto alto; y (iii) a la semana 10, la solicitud formal de un nuevo módulo de geolocalización de puestos fuera del alcance original. Este documento describe el ciclo de vida completo del proyecto —desde el acta de constitución hasta el acta de cierre— y los artefactos formales generados para gestionar cada evento.
